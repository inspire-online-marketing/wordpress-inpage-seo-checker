Check your WordPress SEO: Have a quick overview where to fix current SEO mistakes and have a fully optimized WordPress site.

== Description ==

For people that want a quick way of seeing their SEO mistakes, this plugin is easy to use and lightweight. Get an overview per page what are the common SEO mistakes. This Wordpress plugin implements the Inpage SEO Checker plugin found at https://github.com/StephanSchmidt/inpageseo

== Installation ==

1. Upload the `wordpress-inpage-seo-checker` folder to the `/wp-content/plugins/` directory
1. Activate the 'wordpress-inpage-seo-checker' plugin through the 'Plugins' menu in WordPress
1. Configure the plugin by going to the `Settings` menu that appears in your admin menu where you find 'inpage seo checker'
